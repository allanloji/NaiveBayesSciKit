# Naive Bayes Scikit

Naive Bayes implementation example with scikit-learn

### Prerequisites

scikit-learn requires:

Python (>= 2.7 or >= 3.4)
NumPy (>= 1.8.2)
SciPy (>= 0.13.3)
Pandas(>=0.23.4)


## Running

python NaiveBayesScikit.py
